# gamified-teaching-tools-app
Gamified teaching tool app. This will be updated as the project matures.

Created using Vite + React

Once repository is cloned / pulled, run with 

Working latest latest version of Node.js found here: https://nodejs.org/en

`cd gamified-teaching-tools-app`
`npm run dev`

# Repository Guide:

gamified-teaching-tools-app/
- Folder that contains all the React code.

gamified-teaching-tools-app/src/App.jsx
- This is where the entry point of our code is.
